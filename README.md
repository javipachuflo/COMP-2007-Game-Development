## 💻 How to Run

1.  **Clone** this repository using your method of choice (and remember where you saved it!).
2.  Open **Unity Hub**.
3.  Click **`Add`** (top right) and select **`Add project from disk`**
4.  Go inside the repo's main folder and choose whichever project you like! 
5.  Once it appears under **`Projects`**, run it normally from Unity Hub
