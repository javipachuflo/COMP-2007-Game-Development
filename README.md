# 🎮 COMP 2007: Game Development

This repository contains the work and projects completed during the **COMP 2007** module's lectures and labs.

> 🛠️ **Tech Stack:** These projects were built using **Unity (2021.2.8f1)** and **C#**.

---

## 🚀 How to Run

1. **Clone** this repository using your method of choice (and remember where you saved it!).
2. Open **Unity Hub**.
3. Click **`Add`** (top right) and select **`Add project from disk`**.
4. Navigate inside the cloned repository's main folder and select the project you wish to view.
5. Once it appears under your **`Projects`** list, click it to run it normally.
